# MediCrece — Sistema de Gestión Médica Pediátrica

App web progresiva (PWA) para gestión de expedientes médicos pediátricos.

## Tecnologías
- React 18
- Supabase (base de datos)
- Vercel (hosting)

## Instalación local

```bash
npm install
npm start
```

## Variables de entorno
Configuradas directamente en src/App.jsx:
- SUPABASE_URL
- SUPABASE_KEY
